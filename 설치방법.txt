================================================================
  TechManual Table Extractor — 설치 및 사용 안내 (비개발자용)
================================================================

■ 필요 환경
  - Windows 10/11 (64bit)
  - Python 3.13 (64bit)  ← PC당 최초 1회 설치
  - .hwp 파일 사용 시: 한컴오피스(한글) 설치 필요
  - .hwpx 파일만 사용 시: 한글 설치 불필요

----------------------------------------------------------------
[최초 설치 — PC당 1회]
----------------------------------------------------------------

1) Python 3.13 설치
   - https://www.python.org/downloads/ 에서 Python 3.13 다운로드
   - 설치 시 맨 아래 "Add python.exe to PATH" 반드시 체크
   - "Install Now" 클릭 후 완료

2) 프로그램 설치
   - 이 폴더 전체를 원하는 위치에 복사 (예: C:\Tools\TechManual_Table_Extractor)
   - install_offline.bat 더블클릭
   - "Done" 과 "PyQt6 OK" 메시지가 나오면 설치 완료

   ※ 인터넷 없는 PC에서도 wheelhouse 폴더 덕분에 설치 가능합니다.

----------------------------------------------------------------
[사용 방법 — 매번]
----------------------------------------------------------------

1) run_gui.bat 더블클릭 → 프로그램 창이 열립니다.

2) [파일 선택] — 추출할 .hwp / .hwpx 파일 또는 폴더 선택

3) 처음 사용 시:
   - "모든 표 추출" 선택 → [추출 실행]
   - 결과 엑셀에서 원하는 표의 1행 헤더 확인

4) 규칙 설정:
   - 규칙 표에서 [규칙 추가]
   - 헤더 칸에 "품명, 규격, 수량" 처럼 쉼표로 입력
   - [규칙 저장] 클릭

5) "규칙에 맞는 표만 추출" 선택 → [추출 실행]

6) [결과 폴더 열기]로 엑셀 파일 확인
   - output 폴더에 <파일명>_tables.xlsx 가 생성됩니다.

----------------------------------------------------------------
[자주 나오는 오류]
----------------------------------------------------------------

  "Python 3.13 not found"
    → Python 3.13 미설치 또는 PATH 미체크. Python 재설치.

  "venv not found"
    → install_offline.bat 을 먼저 실행하세요.

  "Install failed"
    → wheelhouse 폴더와 requirements.txt 가 있는지 확인.
       폴더가 불완전하면 개발자에게 전체 폴더를 다시 받으세요.

  GUI 창이 안 뜸
    → install_offline.bat 을 다시 실행해 보세요.

----------------------------------------------------------------
[바로가기 만들기 (선택)]
----------------------------------------------------------------

  run_gui.bat 우클릭 → "바로 가기 만들기"
  → 바로가기를 바탕화면으로 이동
